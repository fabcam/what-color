# What Color 🎨

A Chrome extension that helps you discover color names and hex codes in seconds.

## Features

✨ **Color Name Lookup** - Get the name of any color from its hex code  
🎯 **Color Picker** - Pick colors directly from your screen  
⚡ **Fast & Simple** - Lightweight and easy to use  
🔄 **Real-time Results** - Instant feedback as you explore colors  

## How to Use

1. **Install the extension** from the Chrome Web Store or load it manually
2. **Click the extension icon** in your Chrome toolbar
3. **Enter a hex code** (e.g., `#FF5733`) to get the color name, or
4. **Use the color picker** to select colors from any webpage

## Installation

### From Chrome Web Store
[Link to Chrome Web Store](https://chromewebstore.google.com/detail/what-name/kgmpafilalfdpgphbhmldeplplkjebnc?authuser=2&hl=en)

### Manual Installation (Development)
1. Clone the repository:
   ```bash
   git clone https://github.com/fabcam/what-color.git
   ```
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked** and select the project folder
5. The extension will appear in your Chrome toolbar
